# Zoo Guide Agent

## Overview
The Zoo Guide Agent is an interactive application designed to enhance the visitor experience at zoos by providing real-time information, guiding users through exhibits, and offering educational content.

## Features
- **Interactive Maps**: Navigate through the zoo with ease.
- **Animal Information**: Get detailed information about animals, including their habitat, diet, and behaviors.
- **Event Notifications**: Stay updated on scheduled events, feedings, and talks.
- **User-friendly Interface**: Designed for visitors of all ages.

## Installation
To run the Zoo Guide Agent locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/Jaysarna/zoo_guide_agent.git
   ```
2. Navigate into the directory:
   ```bash
   cd zoo_guide_agent
   ```
3. Install necessary dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application:
```bash
npm start
```
